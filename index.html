What is Chordova?

Chordova is a web-based audio player designed to play audio files you already have on your computer or mobile device. It uses Apache Cordova to target mobile devices as an app (so no internet connection is required), with the possibility of future mobile-specific functionality granted through Cordova plugins, but is also able to run as a static webpage to theoretically be usable on any platform with a web browser that supports HTML5 and JavaScript.
